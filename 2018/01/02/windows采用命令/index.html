常用的windows命令

Web安全
  • windows命令
  • 系统

wmic

wmic是wmi的扩展，由于别名的原因wmic比wmi更加直观。wmic提供了从命令行接口和批命令脚本执行系统管理的支持

直接在cmd中运行wmic即可直接进入一个具有root权限的交互式shell

获取用户信息，在交互界面输入useraccount list brief可以获取详细的用户信息，


识别Windows启动查程序


识别windows网卡
wmic还可以提供有关网卡和驱动程序的大量信息
列出所有网络驱动程序列表
wmic nicconfig list

更新静态ip
mic nicconfig where index=9 call enablestatic("192.168.123.66"), ("255.255.255.0")
更新网关
wmic nicconfig where index=9 call setgateways("192.168.123.4"),(1)
启动DHCP
wmic nicconfig where index=9 call enabledhcp
设置DHCP状态

处理windows进程

列出进程


以xml格式存储

出正在运行的进程并输出到HTML/XSL格式
wmic /output:wmic.html process list full /format:hform

windows用户

显示本地计算机的所有用户账户的列表

显示某个账户的信息
net user someuser

添加test用户
net user test /add
使用24小时制表示法设置test的登录时间
net user test /time:M-F,8AM-5PM

列出事件日志
wmic ntevent list brief
删除arp
netsh int ip delete arpcache
重启和关机

详细查询语句

识别Windows的安全补丁

wmic qfe get description,installedOn

sc命令

作为一个命令行工具，SC.exe可以用来测试你自己的系统，你可以设置一个批处理文件来使用不同的参数调用 SC.exe来控制服务。
SC [Servername] command Servicename [Optionname= Optionvalues]

识别windows的服务

查询正在运行的服务
sc query type=service

查询存在但没有运行的服务
sc query type=service state=inactive

查询所有服务
sc query type=service state=all

网络连接监视

所有侦听端口和可执行文件的刷新率均为3秒
netstat -nab 3 >> netstat.tx
-n：以数字形式显示地址和端口号
-a ：显示所有连接和侦听端口
-b：显示创建每个连接或侦听端口所涉及的可执行文件
