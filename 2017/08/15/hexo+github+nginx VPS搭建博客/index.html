VPS上安装Nginx

首先是在/etc/yum.repos.d目录下创建一个源配置文件nginx.repo,写入如下代码

[nginx]
name=nginx repo
baseurl=http://nginx.org/packages/centos/$releasever/$basearch/
gpgcheck=0
enabled=1

VPS上部署Nginx

在/etc/nginx/sites-available目录下创建一个配置文件XXX.conf(这里以hexo.conf为例),并在文件中输入一下内容

server {
  root /var/www/blog/public; #站点目录
  index index.html index.htm; #主页，不用改
  server_name www.cmustard.com cmustard.com *.cmustard.com; #申请的域名
  location / {
    try_files $uri $uri/ /index.html;
  }
}

执行以下命令，通过vim写入内容

mkdir -p /etc/nginx/sites-available
vim /etc/nginx/sites-available/hexo.conf   #将上面的内容写入到文件中

1.github注册账号
在github官网注册自己的账号，然后创建一个repositorie，假如你github账号名称
为zzz.那就创建一个zzz.github.io的repository。

2.创建ssh key
接下来我们需要在本地创建一个ssh key公钥，并将其添加到github中

此时在~/.ssh/下生成一个id_ras.pub的文件，复制这个文件中的信息，将其添加到github中。

在你github账号的setting中有一个SSH and GPG keys选项，进入这个选项后，在SSH keys栏中New SSH key 自己起一个名字，然后将id_ras.pub中的信息复制到key中，然后ADD。如果没有出现错误，就说明添加成功
