HTMLParser模块

python
  • python
  • 爬虫

简介

HTMLParser实例可以分析和处理在html数据中的遇到开始标签，结束标签，文本，注释和其他标签。我们可以通过重写HTMLParser的子类的方法来实现我们期望的行为
这个HTMLParser实例化是不需要其他参数的

类的定义和常用方法

类的定义

  1. HTMLParser主要用来解析HTML文件
  2. HTMLParser可以接收相应的HTML内容，并进行解析，遇到HTML的标签会自动调用相应的handler（处理方法）来处理，用户需要自己创建相应的子类来继承HTMLParser，并且复写相应的handler方法
  3. HTMLParser不会检查开始标签和结束标签是否是一对

常用方法

HTMLParser.feed(data):接受一个字符串类型的HTML内容，并进行解析
HTMLParser.close():当遇到文件结束标签后进行的处理方法，如果子类要复写该方法，需要首先调用HTMLParser类的close()方法
HTMLParser.reset():重置HTMLParser实例，该方法会丢掉未处理的html内容
HTMLParser.getpos()：返回当前行和相应的偏移量
HTMLParser.handle_starttag(tag,attrs):对开始标签的处理方法，例如<div id="main">，参数tag指的是div，attrs指的是一个(name,value)列表，
HTMLParser.handle_endtag(tag):对结束标签的处理方法。例如</div>,参数tag指的是div
HTMLParser.handle_data(data):对标签之间的数据的处理方法。<tag>test</tag>,data指的是”test”
HTMLParser.handle_comment(data):对HTML中注释的处理方法
HTMLParser.handle_charref(name):处理特殊字符，就是以&#开头的，然后获取&#后面的字符进行进一步，操作一般是内码表示的字符,如&#62,$#x3E
HTMLParser.handle_entityref(name):处理一些特殊字符，以&开头的，然后获取&后面的字符进行进一步操作，如&gt
HTMLParser.handle_decl (decl): 处理<!开头的，比如<!DOCTYPE html PUBLIC “-//W3C//DTD HTML 4.01 Transitional//EN”
HTMLParser.handle_pi(data):处理形如<?instruction>的东西

htmlentitydefs

htmlentitydefs.entitydefs

A dictionary mapping XHTML 1.0 entity definitions to their replacement text in ISO Latin-1.

htmlentitydefs.name2codepoint

A dictionary that maps HTML entity names to the Unicode code points.
将HTML实体映射到Unicode编码的字典

htmlentitydefs.codepoint2name

A dictionary that maps Unicode code points to HTML entity names.
将Unicode实体映射到HTML实体的字典
